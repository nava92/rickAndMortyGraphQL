# rickAndMortyGraphQL
This is the technical test for Multimoney written in Swift

Arquitectura MVVM

El schema para graphQL se ejecuta de manera automática por medio de un script en la Build Phase.
Diferentes métodos para hacer las Queries dentro del código, demostrando varias maneras de hacerlo.
Diferentes maneras de mostrar los resultados, una por medio de un List, y otro por medio de un ScrollView. De igual manera para demostrar diferentes maneras de lograr un objetivo.
Metodología GitFlow
